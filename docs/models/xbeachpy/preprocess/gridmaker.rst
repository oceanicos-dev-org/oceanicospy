Gridmaker
=========

.. toctree::
   :maxdepth: 4

The ``gridmaker`` module is responsible for creating the grid structure for XBeach simulations.
It handles the generation of uniform 2D grids from a shapefile bounding box and ensures
that the output ``.grd`` files are properly formatted and registered in ``params.txt``.
