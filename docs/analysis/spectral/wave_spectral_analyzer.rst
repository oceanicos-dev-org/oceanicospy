Wave Spectral Analyzer
======================

.. toctree::
   :maxdepth: 2

The methods used to analyze waves in the spectral domain are implemented through the `WaveSpectralAnalyzer` class. 
This class provides various functionalities to compute and analyze wave spectra, including methods for calculating the power spectral density, wave energy, and other spectral properties.
