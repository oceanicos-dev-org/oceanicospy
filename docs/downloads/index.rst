Downloads
=========

This document provides an overview of the downloads module in the **oceanicospy** package. 
The downloads module is designed to facilitate the fetching of oceanographic data from various sources.

The following sources are currently supported:

- **UHSLC**: `University of Hawaii Sea Level Center <https://uhslc.soest.hawaii.edu>`_
- **CMDS**: `Copernicus Marine Data Store <https://data.marine.copernicus.eu/products>`_
- **ERA5**: `ECMWF Reanalysis 5th Generation <https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=download>`_
