Support
=======

**oceanicospy** is an open-source project developed and maintained by the
`OCEANICOS developer team <https://github.com/oceanicos-dev-org>`_ at the
Universidad Nacional de Colombia.

Getting help
------------

The easiest way to get help is to open an issue on GitHub:

   https://github.com/oceanicos-dev-org/oceanicospy/issues

When opening an issue, please include:

- A short description of what you are trying to do.
- The version of oceanicospy you are using (``pip show oceanicospy``).
- A minimal, reproducible code example if the issue is a bug.

For questions that are not bugs or feature requests, you can also use GitHub
Discussions:

   https://github.com/oceanicos-dev-org/oceanicospy/discussions

Direct contact
--------------

For specific inquiries you can contact the development team at:

   oceanicos_med@unal.edu.co

Contributing
------------

If you would like to contribute a fix, feature, or documentation improvement,
see the :doc:`Contributing <contributing>` guide for the full workflow.
