Grid
====

.. toctree::
   :maxdepth: 2

``Grid`` constructs the 2D sampling axes of a regular spatial grid.  It
supports constant spacing and can be built from explicit bounding-box
coordinates or from the bounding box of a shapefile.  Once built, the grid
can be exported as plain-text ``.grd`` files for use in numerical models.
