Profile axis
============

.. toctree::
   :maxdepth: 1

``ProfileAxis`` constructs the 1D sampling axis of a straight-line
profile.  It supports constant or piecewise-constant spacing and can be
built from planar endpoint coordinates or from a nominal profile length.
The axis is the primary input for :class:`~oceanicospy.gis.ProfileInterpolator`.
